#  Orçamento do Projeto  
## Pedido Fácil – App de Pedidos para Restaurante Casual

---

##  Visão Geral

Este orçamento foi elaborado de forma **fictícia**, simulando uma proposta real
para o desenvolvimento do aplicativo **Pedido Fácil**.

Os valores refletem o tempo, a complexidade e o nível de detalhamento de cada fase
do projeto.

---

##  Etapas do Projeto


| Briefing & Pesquisa | Levantamento de requisitos e análise do problema | 800,00 |
| Design System | Definição de cores, tipografia e componentes | 1.200,00 |
| Wireframes de Alta Fidelidade | Criação das telas do aplicativo | 1.500,00 |
| Prototipagem | Conexão das telas e fluxos de navegação | 1.000,00 |
| Documentação | Organização do projeto e arquivos no GitHub | 500,00 |

---

##  Valor Total do Projeto

**R$ 5.000,00**

---

##  Prazo Estimado

O prazo estimado para a conclusão do projeto é de aproximadamente:

- **4 a 5 semanas**, considerando todas as etapas descritas.

---

##  Observações

- Este orçamento considera apenas as etapas de **design e prototipagem**
- Não inclui desenvolvimento (frontend ou backend)
